Computer History

Computers have been around for nearly 200 years. The word computer originally referred to humans who performed calculations, like in the movie Hidden Figures. Now it can refer to a machine whose primary purpose is calculating.

File Manipulation

It is possible to create and manipulate files directly in VS code. There are multiple ways to do it. You can use the navigation system to create a file or the command prompt. You can specify the filetype/extension when you name the file.

GitHub

GitHub gives you the ability to collaborate on code with your team. It is considered a distributed version control system.

Markdown Language

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. This week, we used markdown language to format a simple webpage. One of the benefits of Markdown language is that it is less involved than HTML.

Repositories

A repository is a storage space in GitHub where the files for storage and version history are kept. This week the whole class has been adding blogs to the SideQuests repository. We have also been creating our own repositories for various projects.

Version Control

Version Control is a feature in GitHub that allows you to see a history of the changes made to the repository, very similar to the version history in google docs. This makes it easier to identify problems in the code.

Github Pages

Github pages allows you to host your repository on the internet. This makes it easy to retrieve the document at any time.
